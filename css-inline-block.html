Bemerkungen

1. neue / dieselbe Zeile

block-E - neue Zeile, sind so breit, wie möglich (100% vom Elternelement)
inline-E - dieselbe Zeile, sind so breit, wie nötig

2. box-modell

sowohl an inline- als auch block-E kann man das CSS-Box-Modell anwenden

aber margin bei inline-E hat eine Wirkung nur links und rechts

3. Maßangaben

block-E - akzeptieren width & height
inline-E - ignorieren width & height

4. das Verhalten ändern

über die Eigenschaft display kann man block- & inline-Verhalten ändern

5. inline-block

über display:inline-block kann man Vorteile von beiden Verhalten kombinieren
d.h. mehrere Elemente in gleicher Zeile und Maßangaben
